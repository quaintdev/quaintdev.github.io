I Created Entire App in 30 Min With ChatGPT

December 18, 2022 · 7 min · Rohan D

I have been playing around with ChatGPT for quite some time now and have used it for quite a few things like blog posts, generation of thumbnails, simple programs, random research on certain topics, etc,. Now I wanted to see just how useful ChatGPT can be if I am starting a project from scratch. I started working on this project called Echoes of Eternity. The app sounds very fancy but it’s actually very simple. It can

  1. Give you a random quote of the day when you access /qotd endpoint
  2. List all the quotes that are present in database when you hit /quotes endpoint

Let’s dive straight into it. Here goes my first prompt

I need a Go program that will respond to /qotd endpoint with a Quote json. Quote is a struct holding author and a quote.

ChatGPT:

package main

import (
	"encoding/json"
	"net/http"
)

type Quote struct {
	Author string `json:"author"`
	Quote  string `json:"quote"`
}

func main() {
	http.HandleFunc("/qotd", quoteOfTheDay)
	http.ListenAndServe(":8080", nil)
}

func quoteOfTheDay(w http.ResponseWriter, r *http.Request) {
	quote := Quote{
		Author: "Mark Twain",
		Quote:  "Good friends, good books, and a sleepy conscience: this is the ideal life.",
	}

	json, err := json.Marshal(quote)
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}

	w.Header().Set("Content-Type", "application/json")
	w.Write(json)
}

As you can see it knows that qotd is quote of the day and that it will need a Quote struct. Awesome! Now I want quote fetched from database so here goes another prompt

Now update above code to read a random quote from quotes.db

ChatGPT:

func quoteOfTheDay(w http.ResponseWriter, r *http.Request) {
	db, err := sql.Open("sqlite3", "quotes.db")
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}
	defer db.Close()

	var count int
	err = db.QueryRow("SELECT COUNT(*) FROM quotes").Scan(&count)
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}

	var idx = rand.Intn(count)
	var quote Quote
	err = db.QueryRow("SELECT author, quote FROM quotes WHERE id = ?", idx).Scan(&quote.Author, &quote.Quote)
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}

	json, err := json.Marshal(quote)
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}

	w.Header().Set("Content-Type", "application/json")
	w.Write(json)
}

So now our app can at least respond with random quote. Let’s add some more capability to it

Now give me an API that will list all the quotes on certain endpoint

It adds http.HandleFunc("/quotes", listQuotes) in main and then goes on to give implementation of listQuotes

ChatGPT

func listQuotes(w http.ResponseWriter, r *http.Request) {
	rows, err := db.Query("SELECT id, author, quote FROM quotes")
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}
	defer rows.Close()

	var quotes []Quote
	for rows.Next() {
		var quote Quote
		err := rows.Scan(&quote.ID, &quote.Author, &quote.Quote)
		if err != nil {
			http.Error(w, err.Error(), http.StatusInternalServerError)
			return
		}
		quotes = append(quotes, quote)
	}

	json, err := json.Marshal(quotes)
	if err != nil {
		http.Error(w, err.Error(), http.StatusInternalServerError)
		return
	}

	w.Header().Set("Content-Type", "application/json")
	w.Write(json)
}

These are just endpoints we need some front-end to view our quotes

Now give me a HTML homepage that can list all the quotes available at /quotes endpoint

ChatGPT

<html>
<head>
	<title>Quotes</title>
</head>
<body>
	<h1>Quotes</h1>
	<ul id="quote-list"></ul>
	<script>
		fetch("/quotes")
			.then(response => response.json())
			.then(quotes => {
				const quoteList = document.getElementById("quote-list");
				for (const quote of quotes) {
					const li = document.createElement("li");
					li.innerHTML = `"${quote.quote}" - ${quote.author}`;
					quoteList.appendChild(li);
				}
			});
	</script>
</body>
</html>

Along with above HTML the response also included updated Go code to serve above html

	http.HandleFunc("/", homepage)
	http.HandleFunc("/qotd", quoteOfTheDay)
	http.HandleFunc("/quotes", listQuotes)
	http.ListenAndServe(":8080", nil)
}

func homepage(w http.ResponseWriter, r *http.Request) {
	http.ServeFile(w, r, "index.html")
}
